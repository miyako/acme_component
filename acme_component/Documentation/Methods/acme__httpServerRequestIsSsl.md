# **Method :** acme__httpServerRequestIsSsl
## **Scope :** private
## **Treadsafe :** capable ✅ 
## **Description :**

| $0 | OUT | ssl | BOOLEAN | TRUE if connection is Ssl, FALSE otherwise |
